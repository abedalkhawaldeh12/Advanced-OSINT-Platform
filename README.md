## 🔥 Key Features

- **Global Infrastructure Mapping:** DNS (A, MX, TXT), WHOIS lookups, and precise 3D Geo-Tracking for server locations.
- **Deep Web Discovery:** Subdomain Enumeration and Historical Archiving via the Wayback Machine.
- **Offensive Vulnerability Scanning:** 
  - Automated **Subdomain Takeover** vulnerability detection.
  - Identification of exposed data breaches using Dark Web APIs.
  - Target server port scanning and Tech-Stack fingerprinting.
  - Security misconfiguration alerts (Missing Headers, robots.txt exposure).
- **Data Harvesting:** Automated Email extraction and Google Dorks query generation.
- **Professional Reporting:** 1-Click generation of highly structured PDF Intelligence Reports.

## 🛠️ Local Installation

If you want to run the platform locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abedalkhawaldeh12/Advanced-OSINT-Platform.git
   cd Advanced-OSINT-Platform
   ```

2. **Install Backend Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the Platform:**
   ```bash
   uvicorn main:app --reload
   ```

4. Open your browser and go to `http://127.0.0.1:8000`
